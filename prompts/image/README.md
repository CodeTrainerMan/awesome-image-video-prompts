# Image Prompts

This directory contains prompts for AI image generation tools.

## Categories

- **photography/** - Photorealistic photography prompts (landscapes, portraits, street photography, etc.)
- **artistic/** - Artistic and creative prompts (abstract, paintings, digital art, etc.)
- **digital-art/** - Digital art and illustration prompts
- **product/** - Product photography and e-commerce prompts
- **architecture/** - Architecture and interior design prompts
- **fashion/** - Fashion photography and style prompts

## File Naming Convention

Files follow the format: `category-style-tags.md` or `category-style-tags.txt`

Examples:
- `landscape-sunset-golden-hour.md`
- `portrait-photorealistic-studio.md`
- `abstract-colorful-modern.md`

## Usage

1. Browse the categories to find prompts that match your needs
2. Open the prompt file to see the full prompt text and parameters
3. Copy the prompt and customize as needed for your AI tool
4. Check the "Usage Notes" section for tool-specific tips
