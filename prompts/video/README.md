# Video Prompts

This directory contains prompts for AI video generation tools.

## Categories

- **cinematic/** - Cinematic and film-style video prompts (drones, nature, dramatic scenes, etc.)
- **animation/** - Animation prompts (2D, 3D, motion graphics, abstract animations, etc.)
- **commercial/** - Commercial and advertising video prompts (product showcases, brand videos, etc.)
- **social-media/** - Social media video prompts (short-form content, time-lapses, engaging visuals, etc.)
- **educational/** - Educational video prompts (tutorials, explainers, presentations, etc.)

## File Naming Convention

Files follow the format: `category-style-tags.md` or `category-style-tags.txt`

Examples:
- `drone-forest-dawn.md`
- `abstract-particles-flow.md`
- `product-showcase-360.md`
- `urban-time-lapse.md`
- `explainer-animation.md`

## Usage

1. Browse the categories to find prompts that match your needs
2. Open the prompt file to see the full prompt text and creative thinking
3. Copy the prompt and customize as needed for your AI video generation tool
4. Check the "Creative Thinking" section for insights on how the prompt works

## Video-Specific Considerations

When using video prompts, consider:
- **Frame Rate**: Most prompts specify 60fps for smooth motion
- **Duration**: Adjust prompt for desired video length
- **Motion**: Video prompts emphasize camera movement and dynamic elements
- **Loops**: Some prompts are designed for seamless looping
