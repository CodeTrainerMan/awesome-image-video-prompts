## 📖 项目介绍

**Awesome Image & Video Prompts** 是一个公共仓库，用于收集和整理高质量的 AI 图像和视频生成提示词。无论您使用的是 DALL·E、Midjourney、Stable Diffusion 还是其他 AI 生成平台，本仓库都提供了丰富的提示词集合，为您的创意项目提供灵感和支持。

### ✨ 特性

- 🎨 **精选集合**：从各种来源精心挑选的提示词
- 📚 **组织有序**：按风格、主题和用例分类
- 🔄 **持续更新**：定期更新新的提示词
- 🌍 **社区驱动**：欢迎社区贡献
- 📝 **文档完善**：清晰的描述和使用示例

### 📖 使用示例

#### 图像提示词

**基础示例：**
```
夕阳下的宁静风景，黄金时刻光线，
照片级真实感，4K，细节丰富，电影构图
```

**高级示例（Midjourney）：**
```
/imagine prompt: 未来主义城市夜景，赛博朋克风格，
霓虹灯反射在湿润的街道上，电影级光线，
--ar 16:9 --v 6 --style raw
```

**Stable Diffusion 示例：**
```
(杰作, 最佳质量, 超精细), 
美丽的风景, 日落, 黄金时刻,
照片级真实感, 4K, 细节丰富, 电影构图,
负面提示词: 模糊, 低质量, 扭曲
```

#### 视频提示词

**基础示例：**
```
夜晚繁华城市的延时摄影，霓虹灯，
流畅的相机运动，电影级画质，60fps
```

**Runway Gen-2 示例：**
```
电影级无人机镜头，在黎明时分缓慢上升穿越薄雾森林，
揭示隐藏的湖泊，流畅的相机运动，
电影级调色，4K 分辨率
```

### 💡 提示词编写技巧

#### 最佳实践

1. **具体明确**：包含风格、情绪、光线和构图的细节
2. **使用质量标签**：添加如"4K"、"高质量"、"细节丰富"、"杰作"等术语
3. **指定风格**：提及艺术风格（如"照片级真实感"、"动漫"、"油画"）
4. **包含技术细节**：分辨率、宽高比、帧率（视频）
5. **使用负面提示词**：明确不想要的内容（如"无模糊"、"无扭曲"）

#### 常用提示词元素

- **风格**：照片级真实感、艺术风格、卡通、动漫、油画、水彩
- **光线**：黄金时刻、工作室光线、自然光、戏剧性光线
- **构图**：特写、广角、电影级、肖像、风景
- **质量**：4K、8K、高分辨率、细节丰富、清晰对焦
- **情绪**：宁静、戏剧性、充满活力、忧郁、欢快、神秘

### 📚 提示词分类

#### 图像分类
- **艺术风格**：抽象、印象派、超现实主义、极简主义

### 📋 提示词格式规范

为保持一致性，本仓库中的提示词遵循以下标准：

#### 文件命名规范
- 格式：`分类-风格-标签.txt` 或 `分类-风格-标签.md`
- 示例：`portrait-photorealistic-4k.txt`
- 使用小写字母和连字符

#### 提示词结构
```
[标题/描述]
[主要提示词文本]
[可选：负面提示词]
[可选：参数/设置]
[可选：使用说明]
```

#### 元数据标签
- `--ar` 或 `--aspect-ratio`：宽高比（如 16:9, 1:1）
- `--v` 或 `--version`：模型版本
- `--style`：风格修饰符
- `--quality`：质量设置
- `--seed`：随机种子（用于可重现性）
