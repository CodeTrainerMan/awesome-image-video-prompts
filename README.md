### ğŸ’¡ Prompt Writing Tips

#### Best Practices

1. **Be Specific**: Include details about style, mood, lighting, and composition
2. **Use Quality Tags**: Add terms like "4K", "high quality", "detailed", "masterpiece"
3. **Specify Style**: Mention artistic styles (e.g., "photorealistic", "anime", "oil painting")
4. **Include Technical Details**: Resolution, aspect ratio, frame rate (for video)
5. **Use Negative Prompts**: Specify what you don't want (e.g., "no blur", "no distortion")

#### Common Prompt Elements

- **Style**: photorealistic, artistic, cartoon, anime, oil painting, watercolor
- **Lighting**: golden hour, studio lighting, natural light, dramatic lighting
- **Composition**: close-up, wide shot, cinematic, portrait, landscape
- **Quality**: 4K, 8K, high resolution, detailed, sharp focus
- **Mood**: serene, dramatic, vibrant, moody, cheerful, mysterious

### ğŸ“‹ Prompt Format Standards

To maintain consistency, prompts in this repository follow these standards:

#### File Naming Convention
- Format: `category-style-tags.txt` or `category-style-tags.md`
- Example: `portrait-photorealistic-4k.txt`
- Use lowercase with hyphens

#### Prompt Structure
```
[Title/Description]
[Main Prompt Text]
[Optional: Negative Prompt]
[Optional: Parameters/Settings]
[Optional: Usage Notes]
```

#### Metadata Tags
- `--ar` or `--aspect-ratio`: Aspect ratio (e.g., 16:9, 1:1)
- `--v` or `--version`: Model version
- `--style`: Style modifier
- `--quality`: Quality setting
- `--seed`: Random seed for reproducibility
